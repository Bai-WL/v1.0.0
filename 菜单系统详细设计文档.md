# 手持屏控制器菜单系统详细设计文档

## 1. 概述

### 1.1 设计目标
设计一个适用于手持屏控制器（192×128像素显示，六个功能键输入）的轻量级、高效、可扩展的菜单系统。该系统应支持多级菜单导航、多语言、参数编辑、状态显示等功能。

### 1.2 硬件约束
- **显示屏**: JLX192128G (192×128像素，单色)
- **输入设备**: 六个功能键（F1-F6），分别映射为上、下、左、右、Enter、Esc功能
- **处理器**: GD32F303CC (Cortex-M4, 256KB Flash, 48KB RAM)
- **内存限制**: 需优化内存使用，避免动态分配

### 1.3 软件上下文
- 基于现有BSP驱动：显示驱动、HID驱动（funkey）、电源管理等
- 集成到现有应用层框架中
- 利用已有的多语言系统

### 2.2 模块划分

#### 2.2.1 菜单管理器 (MenuManager)
- 菜单树结构定义和维护
- 状态机管理（浏览、编辑、对话框等）
- 导航逻辑处理（前进、后退、跳转）
- 菜单项回调函数管理

#### 2.2.2 显示管理器 (DisplayManager)
- 界面渲染和更新
- 屏幕布局管理
- 字体和图标管理
- 过渡动画和视觉效果

#### 2.2.3 输入处理器 (InputHandler)
- 功能键扫描和事件解码
- 按键消抖处理
- 长按和短按识别
- 输入事件队列管理

```c
    MenuItemType type;            // 菜单项类型

        // ACTION类型
        struct {
            void (*action_callback)(void);  // 动作回调函数
        };

            bool *toggle_value;              // 开关值指针
            void (*toggle_changed)(bool);    // 值改变回调
        };
        
        // VALUE类型
        struct {
            int32_t *value_ptr;              // 数值指针
            int32_t min_value;               // 最小值
            int32_t max_value;               // 最大值
            int32_t step_value;              // 步进值
            void (*value_changed)(int32_t);  // 值改变回调
        };
        
        // LIST类型
        struct {
            uint8_t *selection_ptr;          // 选择索引指针

            uint8_t option_count;            // 选项数量

            uint16_t target_menu_id;         // 目标菜单ID
```

## 6. 显示渲染设计

### 6.1 屏幕布局计算
```c
// 基于192x128像素显示屏的布局
static const ScreenLayout default_layout = {
    .header_height = 16,      // 标题栏16像素
    .menu_area_top = 20,      // 菜单区域从20像素开始
    .menu_area_height = 88,   // 菜单区域高度88像素
    .footer_height = 24,      // 底部状态栏24像素
    .item_height = 16,        // 每个菜单项16像素
    .font_size = 8,           // 8号字体
    .cn_char_width = 13,      // 中文字符宽度13像素
    .cn_char_height = 16,     // 中文字符高度16像素
    .en_char_width = 7,       // 英文字符宽度7像素
    .en_char_height = 16,     // 英文字符高度16像素
    .icon_size = 12,          // 图标12x12像素
    .padding = 2,             // 内边距2像素
    .max_text_width = 100,    // 文本区域最大宽度（像素）- 根据布局计算
    .max_value_width = 50,    // 附加信息区域宽度（像素）
    .line_height = 16,        // 行高16像素
    .max_lines_per_item = 2   // 每个菜单项最多显示2行
};

// 计算可视项数量（考虑多行文本）
#define VISIBLE_ITEMS (default_layout.menu_area_height / default_layout.item_height)  // 5项

// 文本宽度计算辅助函数
static uint16_t calculate_text_width(const char* text) {
    uint16_t width = 0;
    while (*text) {
        // 简单判断：ASCII字符（英文）用英文宽度，其他用中文宽度
        // 实际实现可能需要更准确的字符编码判断
        if ((uint8_t)*text < 128) {
            width += default_layout.en_char_width;
        } else {
            width += default_layout.cn_char_width;
        }
        text++;
    }
    return width;
}

// 自动换行函数：将文本分割为多行，确保每行不超过最大宽度
// 返回实际使用的行数（1或2）
static uint8_t wrap_text_lines(char line1[32], char line2[32], const char* text, uint16_t max_width) {
    uint16_t current_width = 0;
    uint16_t char_count = 0;
    uint8_t line_index = 0;
    uint16_t last_space_pos = 0;  // 最后一个空格位置
    uint16_t last_space_width = 0;
    const char* src = text;
    char* current_line = line1;
    
    while (*src && line_index < 2) {
        uint16_t char_width = ((uint8_t)*src < 128) ? default_layout.en_char_width : default_layout.cn_char_width;
        
        // 检查添加当前字符后是否会超出宽度
        if (current_width + char_width > max_width) {
            // 如果已经超过一行，需要换行
            if (line_index == 0) {
                // 第一行结束
                if (last_space_pos > 0) {
                    // 在最后一个空格处换行
                    current_line[last_space_pos] = '\0';
                    src = text + last_space_pos + 1; // 跳过空格
                } else {
                    // 没有空格，在当前字符处截断
                    current_line[char_count] = '\0';
                }
                
                // 准备第二行
                line_index = 1;
                current_line = line2;
                char_count = 0;
                current_width = 0;
                last_space_pos = 0;
                last_space_width = 0;
                
                // 重新开始处理（src已更新）
                continue;
            } else {
                // 已经是第二行，无法再换行，截断并添加省略号
                if (char_count > 0) {
                    // 回退一个字符，为省略号留出空间
                    char_count--;
                    current_width -= ((uint8_t)*(src-1) < 128) ? default_layout.en_char_width : default_layout.cn_char_width;
                    
                    // 添加省略号
                    if (current_width + default_layout.en_char_width * 3 <= max_width) {
                        current_line[char_count++] = '.';
                        current_line[char_count++] = '.';
                        current_line[char_count++] = '.';
                    }
                }
                break;
            }
        }
        
        // 记录空格位置，用于在单词边界换行
        if (*src == ' ' || *src == '\t') {
            last_space_pos = char_count;
            last_space_width = current_width;
        }
        
        current_line[char_count++] = *src;
        current_width += char_width;
        src++;
    }
    
    // 设置当前行结束
    if (line_index == 0) {
        current_line[char_count] = '\0';
        line2[0] = '\0';  // 第二行为空
        return 1;
    } else {
        current_line[char_count] = '\0';
        return 2;
    }
}

// 文本布局结构，用于存储换行后的文本
typedef struct {
    char line1[32];        // 第一行文本
    char line2[32];        // 第二行文本（如果有）
    uint8_t line_count;    // 实际行数（1或2）
    uint8_t total_height;  // 总高度（像素）
} TextLayout;

// 计算文本布局
static void calculate_text_layout(TextLayout* layout, const char* text, uint16_t max_width) {
    layout->line_count = wrap_text_lines(layout->line1, layout->line2, text, max_width);
    layout->total_height = layout->line_count * default_layout.line_height;
}

// 检查文本是否需要换行显示
static bool needs_wrapping(const char* text, uint16_t max_width) {
    return calculate_text_width(text) > max_width;
}
```

### 7.1 按键扫描和事件生成
```c
// 按键状态机
typedef struct {
    uint8_t raw_key;           // 原始键值（1-6）
    uint8_t debounced_key;     // 消抖后键值
    uint32_t press_time;       // 按下时间
    uint32_t release_time;     // 释放时间
    bool is_pressed;           // 是否按下状态
    bool is_long_press;        // 是否长按
    bool is_repeat;            // 是否重复触发
} KeyState;

// 按键配置
typedef struct {
    uint32_t debounce_time;    // 消抖时间（ms）
    uint32_t long_press_time;  // 长按时间（ms）
    uint32_t repeat_delay;     // 重复触发延迟（ms）
    uint32_t repeat_interval;  // 重复触发间隔（ms）
} KeyConfig;

// 默认按键配置
static const KeyConfig default_key_config = {
    .debounce_time = 20,       // 20ms消抖
    .long_press_time = 1000,   // 1秒长按
    .repeat_delay = 500,       // 长按500ms后开始重复
    .repeat_interval = 100     // 每100ms重复一次
};

// 按键状态数组（6个键）
static KeyState key_states[6];

// 按键扫描状态机
static void key_scan_state_machine(uint8_t key_index, uint8_t raw_state) {
    KeyState* state = &key_states[key_index];
    uint32_t current_time = bsp_get_systick();
    
    switch (state->is_pressed) {
        case false: // 未按下状态
            if (raw_state) { // 检测到按下
                state->press_time = current_time;
                state->is_pressed = true;
                state->is_long_press = false;
                state->is_repeat = false;
            }
            break;
            
        case true: // 按下状态
            if (!raw_state) { // 检测到释放
                state->release_time = current_time;
                state->is_pressed = false;
                
                // 生成短按事件
                if (!state->is_long_press) {
                    InputEvent event = {
                        .type = KEY_EVENT_PRESS,
                        .key = key_index + 1, // 转换为1-6
                        .timestamp = current_time
                    };
                    input_queue_push(&event);
                }
            } else { // 保持按下
                uint32_t press_duration = current_time - state->press_time;
                
                // 检查是否达到长按时间
                if (!state->is_long_press && press_duration >= default_key_config.long_press_time) {
                    state->is_long_press = true;
                    
                    // 生成长按事件
                    InputEvent event = {
                        .type = KEY_EVENT_LONG_PRESS,
                        .key = key_index + 1,
                        .timestamp = current_time
                    };
                    input_queue_push(&event);
                }
                
                // 检查是否开始重复触发
                if (state->is_long_press && press_duration >= default_key_config.long_press_time + default_key_config.repeat_delay) {
                    uint32_t repeat_start = state->press_time + default_key_config.long_press_time + default_key_config.repeat_delay;
                    uint32_t repeat_count = (current_time - repeat_start) / default_key_config.repeat_interval;
                    
                    if (repeat_count > state->repeat_count) {
                        state->repeat_count = repeat_count;
                        state->is_repeat = true;
                        
                        // 生成重复事件
                        InputEvent event = {
                            .type = KEY_EVENT_REPEAT,
                            .key = key_index + 1,
                            .timestamp = current_time
                        };
                        input_queue_push(&event);
                    }
                }
            }
            break;
    }
}

// 按键扫描主函数（在1ms任务中调用）
void input_key_scan(void) {
    // 获取原始按键状态
    uint8_t raw_key = hid_get_funkey();
    
    // 扫描每个按键
    for (uint8_t i = 0; i < 6; i++) {
        uint8_t key_pressed = (raw_key == (i + 1)) ? 1 : 0;
        key_scan_state_machine(i, key_pressed);
    }
}
```

### 7.2 事件队列
```c
// 事件队列结构
#define EVENT_QUEUE_SIZE 16

typedef struct {
    InputEvent events[EVENT_QUEUE_SIZE];
    uint8_t head;
    uint8_t tail;
    uint8_t count;
} EventQueue;

static EventQueue input_queue;

// 初始化事件队列
void input_queue_init(void) {
    input_queue.head = 0;
    input_queue.tail = 0;
    input_queue.count = 0;
}

// 向队列添加事件
bool input_queue_push(InputEvent* event) {
    if (input_queue.count >= EVENT_QUEUE_SIZE) {
        return false; // 队列满
    }
    
    input_queue.events[input_queue.tail] = *event;
    input_queue.tail = (input_queue.tail + 1) % EVENT_QUEUE_SIZE;
    input_queue.count++;
    return true;
}

// 从队列获取事件
bool input_queue_pop(InputEvent* event) {
    if (input_queue.count == 0) {
        return false; // 队列空
    }
    
    *event = input_queue.events[input_queue.head];
    input_queue.head = (input_queue.head + 1) % EVENT_QUEUE_SIZE;
    input_queue.count--;
    return true;
}

// 检查队列是否为空
bool input_queue_is_empty(void) {
    return input_queue.count == 0;
}
```

## 11. 性能优化考虑

### 11.1 内存优化
1. **静态分配**: 所有菜单项使用静态数组，避免动态内存分配
2. **结构体打包**: 使用`#pragma pack(1)`减少结构体填充
3. **字符串共享**: 使用StringID索引而不是字符串拷贝
4. **显示缓存**: 只重绘变化的部分，避免全屏刷新

### 11.2 处理器优化
1. **事件队列**: 使用环形缓冲队列处理输入事件
2. **惰性渲染**: 只在需要时重绘，减少CPU使用
3. **状态缓存**: 缓存计算结果，避免重复计算
4. **整数运算**: 尽量使用整数运算，避免浮点数

### 11.3 电源管理
1. **动态刷新率**: 空闲时降低刷新频率
2. **睡眠模式**: 长时间无操作时进入睡眠
3. **背光控制**: 根据环境光调节背光亮度
4. **按键唤醒**: 支持按键唤醒睡眠状态
