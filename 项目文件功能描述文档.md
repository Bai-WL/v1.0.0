# 手持屏控制器项目文件功能描述文档

## 项目概述
本项目是基于GD32F303CC微控制器的手持屏控制器，支持两种硬件版本：
1. **WiFi版本（带锂电池）**：集成ATBM6431 WiFi模块，支持TCP协议通信
2. **有线版本（外接电源）**：无WiFi模块，外接电源供电

## 目录结构说明

### 1. Drivers/gd32f303xx_bsp - 板级支持包（Board Support Package）

#### 1.1 核心管理文件
- **bsp.c / bsp.h** - BSP核心框架
  - `bsp_RunPer1ms()` - 1ms周期性任务（RTC更新、系统时间刷新、HID按键扫描）
  - `bsp_RunPer10ms()` - 10ms周期性任务（ADC采样、VBUS检测、电源管理、休眠/关机按键检测）

- **bsp_gd32f303xx.c / bsp_gd32f303xx.h** - GD32F303xx系列通用BSP

- **bsp_types.h** - BSP类型定义和通用数据结构

#### 1.2 系统基础模块
- **bsp_assert.c / bsp_assert.h** - 断言系统，用于调试和错误检测

- **bsp_malloc.c / bsp_malloc.h** - 内存分配管理，提供动态内存分配功能

- **bsp_fifo.c / bsp_fifo.h** - FIFO队列实现，用于串行数据缓冲

- **bsp_stamach.c / bsp_stamach.h** - 状态机框架，用于复杂状态管理

- **bsp_gd32f303xx_systick.c / bsp_gd32f303xx_systick.h** - 系统滴答定时器
  - `bsp_SysTickInit()` - 系统滴答定时器初始化（1ms中断）
  - `bsp_SysTimDelayCallback()` - SysTick中断回调函数（仅更新系统滴答计数，应用层时间任务已移到主循环）
  - `bsp_get_systick()` / `bsp_delay()` - 系统时间获取和延时函数

#### 1.3 显示驱动模块
- **bsp_gd32f303xx_jlx192128g.c / bsp_gd32f303xx_jlx192128g.h** - JLX192128G液晶屏驱动
  - `bsp_JLXLcdInit()` - 液晶屏初始化
  - `bsp_JLXLcdClearScreen()` - 清屏操作
  - 支持SPI接口的192×128像素显示控制

- **bsp_font.h** - 字体数据定义

#### 1.4 输入设备驱动
- **bsp_gd32f303xx_knob.c / bsp_gd32f303xx_knob.h** - 正交编码器旋钮驱动
  - `bsp_knob_init()` - 旋钮初始化
  - 旋钮方向检测和脉冲计数

- **drv_hid.c / drv_hid.h** - HID（人机接口设备）驱动
  - `hid_init()` - HID设备初始化
  - `hid_scan()` - HID按键扫描
  - 支持矩阵键盘和功能键输入

#### 1.5 通信模块
- **Modbus通信**
  - **bsp_gd32f303xx_modbus.c / bsp_gd32f303xx_modbus.h** - Modbus协议栈核心
    - `bsp_ModbusInit()` - Modbus初始化
    - 支持RTU/TCP协议，主从模式
  - **bsp_gd32f303xx_mbboard.c / bsp_gd32f303xx_mbboard.h** - Modbus板级配置
  - **bsp_gd32f303xx_mbuser.c / bsp_gd32f303xx_mbuser.h** - Modbus用户回调函数
  - **bsp_modbus_tcp.c / bsp_modbus_tcp.h** - Modbus TCP协议实现
    - `bsp_modbus_tcp_window_init()` - TCP窗口初始化

- **WiFi通信**
  - **bsp_atbm6431.c / bsp_atbm6431.h** - ATBM6431 WiFi模块驱动
    - `bsp_atbm6431_init()` - WiFi模块初始化
    - 支持AT指令集通信
  - **bsp_wifi.c / bsp_wifi.h** - WiFi状态管理
    - `bsp_wifi_dev_init()` - WiFi设备初始化
    - WiFi连接状态机（热点连接、TCP连接等）
  - **bsp_wifi_trans.c / bsp_wifi_trans.h** - WiFi数据传输
    - `bsp_wifi_driver_init()` - WiFi驱动初始化
    - 数据传输状态管理
  - **bsp_wifi_tcp.c / bsp_wifi_tcp.h** - WiFi TCP通信封装

- **USB通信**
  - **bsp_drdusb.c / bsp_drdusb.h** - DRD（双角色设备）USB驱动
    - `bsp_drdusb_init()` - DRD USB初始化
  - **bsp_usbd_init()** - 标准USB设备初始化（在bsp_drdusb.c中）

#### 1.6 硬件抽象层
- **bsp_gd32f30x_adc.c / bsp_gd32f30x_adc.h** - ADC驱动
  - `bsp_adc_init()` - ADC初始化
  - 用于电池电压检测

- **bsp_gd32f30x_fmc.c / bsp_gd32f30x_fmc.h** - 闪存存储器控制器驱动

- **bsp_gd32f30x_rcu.c / bsp_gd32f30x_rcu.h** - 复位和时钟单元驱动
  - `bsp_reinit_rcu()` - RCU重新初始化

- **bsp_gd32f30x_rtc.c / bsp_gd32f30x_rtc.h** - 实时时钟驱动
  - `bsp_RTC_Init()` - RTC初始化
  - `bsp_RTCRunPer1ms()` - RTC实时更新

- **bsp_i2c_gd32f30x.c / bsp_i2c_gd32f30x.h** - I2C总线驱动

#### 1.7 电源管理
- **drv_pm.c / drv_pm.h** - 电源管理驱动（核心）
  - `pm_power_on()` - 电源上电初始化（包含充电功能检测）
  - `pm_handler()` - 电源管理状态机处理
  - `pm_shutdown()` - 关机处理
  - `pm_suspend_request()` - 休眠请求
  - 支持锂电池充电管理、低电量保护、自动休眠/关机

- **bsp_hhctl_pwr.c / bsp_hhctl_pwr.h** - 手持控制器电源管理

#### 1.8 特定硬件控制
- **bsp_gd32f30x_hhctl.c / bsp_gd32f30x_hhctl.h** - 手持控制器硬件初始化
  - `bsp_hhctl_init()` - 手持控制器整体初始化
    - 中断优先级配置
    - HID设备初始化
    - 系统滴答定时器初始化
    - ADC初始化
    - 电源管理初始化
    - 显示初始化
    - WiFi复位引脚配置（仅WiFi版本）

#### 1.9 其他文件
- **bll_bgp.c / bll_bgp.h** - 背景处理逻辑（功能待明确）
- **dev_memo.md** - 开发备忘录
- **modbus函数使用.doc** - Modbus函数使用文档
- **version.md** - 版本信息

### 2. User - 用户层代码

#### 2.1 主程序
- **main.c** - 主程序入口
  - `main()` - 程序入口函数
  - `System_Init()` - 系统初始化
  - `Main_Loop()` - 主循环（电源管理和低功耗）
  - 支持WIFI_H02W宏区分硬件版本

#### 2.2 中断服务程序
- **gd32f30x_it.c / gd32f30x_it.h** - 中断服务程序
  - `SysTick_Handler()` - 系统滴答中断处理（调用bsp_SysTimDelayCallback）
  - `TIMER2_IRQHandler()` / `TIMER3_IRQHandler()` / `TIMER4_IRQHandler()` - 定时器中断
  - `USBD_LP_CAN0_RX0_IRQHandler()` - USB低功耗/CAN0 RX0中断

#### 2.3 硬件配置
- **gd32f30x_libopt.h** - GD32F30x库选项头文件
  - 包含所有必要的GD32F30x标准外设库头文件
  - 根据芯片型号选择包含ENET驱动（仅GD32F30X_CL系列）

- **usbd_conf.h** - USB设备配置
- **usbd_hw.h** - USB硬件抽象层

### 3. Application - 应用层框架

#### 3.1 核心模块（Application/Core/）
- **目录为空** - 计划放置系统初始化、任务调度、状态管理等核心应用代码

#### 3.2 显示菜单（Application/Dispaly_menu/）
- **screen_ids.h** - 屏幕ID定义文件
  - 定义所有屏幕的唯一ID枚举
  - 支持最多10个屏幕界面管理

#### 3.3 按键控制（Application/Key_control/）
- **目录为空** - 计划放置按键事件处理、输入管理等代码

#### 3.4 RS485通信（Application/Rs485_rtu/）
- **device_storage.c / device_storage.h** - PLC装置存储系统
  - 位装置存储：0-16383共16384个布尔值，使用位数组压缩存储
  - 字装置存储：2914-12913共9999个16位整数，使用数组直接存储
  - 提供统一的访问接口，支持地址映射、数据读写和变更通知
- **poll_manager.c / poll_manager.h** - 轮询管理器
  - 管理最多10个界面的地址轮询
  - 支持界面激活/注销时的地址注册管理
  - 每个界面最多64个地址
- **user_mb_controller.c / user_mb_controller.h** - Modbus控制器
- **user_poll.c / user_poll.h** - 用户轮询功能

#### 3.5 RS485 TCP通信（Application/Rs485_tcp/）
- **user_variables.c / user_variables.h** - 用户变量定义
  - 应用层变量存储和管理
  - 计划用于Modbus TCP应用层数据管理

### 6. 功能模块依赖关系

```
主程序流程：
main.c 
  → System_Init()
    → bsp_hhctl_init() 
      → bsp_gd32f30x_hhctl.c中的初始化
        → hid_init()      [drv_hid.c]
        → bsp_SysTickInit() [bsp_gd32f303xx_systick.c]
        → bsp_adc_init()    [bsp_gd32f30x_adc.c]
        → pm_power_on()     [drv_pm.c]
        → bsp_JLXLcdInit()  [bsp_gd32f303xx_jlx192128g.c]
    
  → Main_Loop()
    → pm_handler()          [drv_pm.c]
    → __WFI()低功耗等待

中断处理：
SysTick_Handler() [gd32f30x_it.c]
  → bsp_SysTimDelayCallback() [bsp_gd32f303xx_systick.c]
    → bsp_RunPer1ms()  [bsp.c]
    → bsp_RunPer10ms() [bsp.c]
```

### 7. 硬件版本差异

#### 7.1 WiFi版本（WIFI_H02W宏定义）
- 包含WiFi模块初始化（bsp_atbm6431_init）
- 支持Modbus TCP通信（bsp_modbus_tcp_window_init）
- 锂电池充电管理（在pm_power_on中实现）
- GPIOB Pin2用作WiFi模块复位引脚

#### 7.2 有线版本（无WIFI_H02W宏定义）
- 无WiFi相关代码
- 外接电源供电
- 带纽扣电池保持RTC时钟
- 电源管理功能简化
